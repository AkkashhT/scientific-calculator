# 🧮 Scientific Calculator

A modern dark-themed Scientific Calculator built using Python and Streamlit.

---

## 📌 Features

- Dark modern UI
- Scientific calculator operations
- Square button layout
- Backspace support
- History section
- Responsive design
- Built using Streamlit

---

## 🛠️ Technologies Used

- Python
- Streamlit
- CSS styling inside Streamlit

---

## 📂 Project Structure

```bash
scientific-calculator/
│
├── app.py
├── requirements.txt
├── .gitignore
└── README.md
```

---

## ▶️ Run Locally

### 1 Clone Repository

```bash
git clone https://github.com/AkkashhT/scientific-calculator.git
```

### 2 Open Project Folder

```bash
cd scientific-calculator
```

### 3 Install Requirements

```bash
pip install -r requirements.txt
```

### 4 Run App

```bash
streamlit run app.py
```
